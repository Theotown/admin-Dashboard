# admin-Dashboard

A responsive, grid-based admin dashboard built with HTML, CSS (Grid & Media Queries), and JavaScript. This project demonstrates modern layout techniques, dark/light theming, responsiveness, and localStorage persistence Ideal for portfolio projects or learning front-end fundamentals.

## ✨ Features

🧱 CSS Grid layout for the full-page structure

📱 Responsive design using media queries

🌙 Dark mode toggle with theme persistence via localStorage

⚡ Dynamic project cards generated with JavaScript

🎨 Clean UI with Material Design Icons and Roboto from Google Fonts

## 📁 Project Structure

 dashboard-project/

├── CSS/
│   └── style.css         # Main CSS for layout and styling

├── Icon/                 # SVGs or PNGs for icons

├── Script/
